# PID Register

This page is the home, or root, page of the `linked.data.gov.au` main
Register.

This *Register* lists other *Register* instances, one for each of the
classes of things that the Australian govenrment Linked Data Working
Group (AGLDWG) issues Persistent Identifiers (PIDs) for. They are:

- [Definitions](http://linked.data.gov.au/def) - ontologies,
  vocabularies and so on (resource-defining things)
- [Datasets](http://linked.data.gov.au/dataset) - Linked Data datasets
- [Organisations](http://linked.data.gov.au/org) - AGLDWG organisations
